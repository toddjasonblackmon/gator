gator is a command line RSS aggregator.

It requires a Postgres database to run the program.
Instructions for installation are available at: https://www.boot.dev/lessons/d049abb2-c733-4986-a33a-73f05e2ea06c

In addition, a database Migration tool, goose is needed to initialize the database.
Instructions for installation and use are available at: https://www.boot.dev/lessons/d049abb2-c733-4986-a33a-73f05e2ea06c

Futher instructions can be found in the guided project on Boot.dev.
